HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements can have attributes.
  • Elements can have classes and ids.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used to style webpages.

Git

  • Git is a software used for version control.
  • Git allows developers to come back to different versions of their project.
  • Git tracks changes in the source code by giving each change a unique identifier number, called SHA.

JavaScript

  • JavaScript is a programming language.
  • JavaScript allows you to make web pages interactive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can give class attributes to HTML elements.
